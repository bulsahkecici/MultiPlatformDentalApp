# Ekran Görüntüleri Klasörü

Bu klasör, `docs/PROJE_DOKUMANTASYONU.md` dosyasında referans verilen ekran görüntülerini içerir.

## Nasıl eklenir?

1. Uygulamayı ilgili platformda çalıştırın.
2. Ekranı **1280×720** veya **1920×1080** çözünürlükte yakalayın.
3. PNG formatında, aşağıdaki dosya adlarıyla bu klasöre kaydedin.
4. Mümkünse demo hesaplardan biriyle giriş yapın (ör. `sekreter@mail.com`).

## İpuçları

- Kişisel veya gerçek hasta verisi göstermeyin; demo veri (`npm run db:seed:demo`) kullanın.
- Windows'ta **Win + Shift + S** ile ekran görüntüsü alınabilir.
- Görüntüleri eklemeden önce hassas bilgileri (gerçek TC, telefon) bulanıklaştırın.
